# Miwok
Android | Create an App to Learn the Miwok Language   
This includes:   
- Building Layouts
- Intents and Activities
- Arrays, Lists, Loops & Custom Classes
- Images and Visual Polish
- Activity Lifecycle and Audio Playback
- Fragments
